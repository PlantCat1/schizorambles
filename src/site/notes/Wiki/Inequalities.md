## Addition Property
If $a~\textless~b$, then $a+c~\textless~b+c$.
$a~\textless~b~\Rightarrow~a+c~\textless~b+c$.
This says that we can add $c$ to both sides of an inequality without changing the inequality's direction.
$\forall~a,b,c\in\mathbb{R}$
Example:
$a=2,~b=3,~c=2$
For $2~\textless~3$, add $2$, to both sides.
$2+2~\textless~3+2$
$4~\textless~5$
Example:
$x-2~\textless~7$
$x-2+2~\textless~7+2$
$x~\textless~9$
